To Run specific problem instance:

1. Go to DeliveryRoutes/Main.java and specify the filename of problem instance (edinburghTest/iowa_1/iowa_2/iowa_3/iowa_4). 
For edinburgh osmfile will be "scot_export.pbf" and for Iowa instances "iowa_export.pbf".

To Run algorithms, go to EvolutionaryAlgorithm Folder.

Original Algo: Baseline_Algorithm.java
Proposed Algo: my_algorithm_final.java
NSGA2: MDVRP_NSGA2_modified.java
NSGA3: MDVRP_NSGA3_modified.java
Without Mutation: my_algorithm_without_mutation.java
Without Taboo List: my_algorithm_without_taboolist.java

Run specific java file. Corresponding csv fil will be generated with the results, as mentioned in Run_Comparison.java.

Two compare, run Run_Comparison.java with specified file names.
